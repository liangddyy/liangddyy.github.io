#### 模块安装失败解决

手动从这里下载 ` http://www.lfd.uci.edu/~gohlke/pythonlibs/` 下载目录执行 ` pip install xxx.whl`安装

#### no module named win32api

`pip install pypiwin32`

#### no module named MySqlDB
